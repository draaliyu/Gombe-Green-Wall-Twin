How to use this page

Choose any LGA from the map or list. Northern LGAs receive NDVI and cellular-grid summaries where the current analysis grid intersects their boundaries.

LGA summaries aggregate available pixels and model cells inside each boundary. Weather is queried at the LGA centroid. Values labelled demo are not observations.
